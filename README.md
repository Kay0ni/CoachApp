## 📖 Le Lore (Description du projet)
**CoachApp** est un outil de diagnostic développé en **C#**[cite: 4]. 
Sa mission principale est de calculer l'**IMC** (Indice de Masse Corporelle) de l'utilisateur pour évaluer ses statistiques de forme physique avec précision[cite: 4]. 

L'application utilise la formule mathématique standard pour déterminer si votre "LV" de santé est optimal :
$$IMC = \frac{poids}{taille^2}$$

## ⚔️ Armes Équipées (Fonctionnalités)
* **[ACT] Saisie de données :** Entrez votre poids (kg) et votre taille (m) via une interface mobile intuitive[cite: 4].
* **[ITEM] Calcul instantané :** L'application traite les données en temps réel grâce à la puissance de .NET[cite: 4].
* **[CHECK] Diagnostic :** Affiche le résultat de l'IMC et interprète la catégorie associée (corpulence normale, surpoids, etc.)[cite: 4].
* **Interface Pixelisée :** Conçue pour rester cohérente avec l'esthétique rétro du portfolio[cite: 1].
