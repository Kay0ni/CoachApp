## 📖 LE LORE (DESCRIPTION DU PROJET)
---------------------------------------------------------
CoachApp est un outil de diagnostic développé en **C#**. 
Sa mission principale est de calculer l'**IMC** (Indice de Masse Corporelle) de l'utilisateur pour évaluer ses statistiques de forme physique avec précision. 

L'application utilise la formule mathématique standard pour déterminer si votre "LV" de santé est optimal :

> IMC = poids / (taille * taille)

## ⚔️ ARMES ÉQUIPÉES (FONCTIONNALITÉS)
---------------------------------------------------------
* **[ACT] Saisie de données :** Entrez votre poids (kg) et votre taille (m) via une interface mobile intuitive.
* **[ITEM] Calcul instantané :** L'application traite les données en temps réel grâce à la puissance de .NET.
* **[CHECK] Diagnostic :** Affiche le résultat de l'IMC et interprète la catégorie associée (corpulence normale, surpoids, etc.).
* **Interface Pixelisée :** Conçue pour rester cohérente avec l'esthétique rétro du portfolio.
