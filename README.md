# 📱 Coach App - [ Évolution des Statistiques ]

* **Zone (Technologies) :** C# / .NET 8.0 (Android & iOS)

## 📖 LE LORE (DESCRIPTION DU PROJET)
---------------------------------------------------------
**Coach App** est une application mobile cross-platform développée avec **Visual Studio 2022**. Sa mission est de calculer l'**IMG** (Indice de Masse Grasse) en fonction du sexe, de la taille, du poids et de l'âge de l'utilisateur. 

L'aventure de son développement s'est déroulée en trois étapes majeures, augmentant progressivement le niveau de complexité et de personnalisation du logiciel.

## ⚔️ HISTORIQUE DES VERSIONS (EXP+)
---------------------------------------------------------

### 🟢 Version 1.0 : La Naissance
* Implémentation de l'algorithme mathématique de calcul de l'IMG en pourcentage.
* Interface de saisie pour les statistiques vitales (Sexe, Taille, Poids, Age).

### 🟡 Version 2.0 : Feedback Visuel
* Ajout d'une **image de réaction** et d'un **commentaire écrit** adaptés au résultat.
* Personnalisation de l'expérience utilisateur selon le statut de forme physique détecté.

### 🔴 Version 3.0 : La Sauvegarde
* Ajout d'un système de **persistance des données**.
* Sauvegarde automatique des derniers calculs pour permettre au joueur de suivre son évolution.
